## Advent of Code

In here I collect some of my solutions for Advent of Code. In 2024 I solved most tasks in C++
There is no real structure and most folders have been compiled as standalone programs. In the beginning I set up a 
unique cmakelists for each folder structure - but since most calculations are generally just a single file I now mostly compile 
with gcc directly.
